HTML

  • The head element contains information or metadata, that helps the browser to render the page correctly. Elements included in the head are not visible when the user looks at the webpage.
  • The body element contains the content that is shown to the user. All the elements that we want the user to see and interact with.

CSS

  • The most common way to style a webpage is by using external CSS style sheet. Other options include, inline CSS and internal style sheet.
  • The external CSS style sheet is preferred because it allows developers to keep all of their CSS rules in a seperate file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are curently on.
  • git checkout -b branch-name: creates a new feature branch.
  • git checkout main: to swithc to main branch.
  • start (name): open a file.
  • touch (new name): creates a new file in a folder.
  • git add -A: To stage changes.
  • git commit -m "your message here": to add commit message.
  • git pull origin main: pull remote main branch into local main branch.
  • git push origin (branch name): to push changes to remote repository.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • To access Chrome.Dev tools, Control + Shift + I
  • 2 requirements of JavaScript functions: Define the function, then Call the function!
